# Git-Example

### Commands

      root@630a1f8f0dcd 
   
- root คือ user 

- 630a1f8f0dcd คือ hostname

#### echo

คำสั่งสำหรับใช้แสดงข้อความ

      echo $SHELL
      echo $0
      
- ใช้ $ เพื่อระบุ system variable ที่ชื่อ SHELL

#### uname

คำสั่งที่พิมพ์ข้อมูลพื้นฐานเกี่ยวกับชื่อระบบปฏิบัติการและฮาร์ดแวร์ระบบ

      uname
      uname -a
